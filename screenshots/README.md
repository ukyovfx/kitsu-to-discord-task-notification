## Capture Notes

- remove secrets, webhook URLs, cookies, and runtime identifiers
- avoid showing private studio names if sharing publicly
- prefer desktop-width screenshots for the first public release
- if you add mobile captures, label them clearly
- do not commit screenshots until they are scrubbed for release
